## 🛠️ Technical Deep Dive

### 1. Data Exploration & Visualization (Task 01)
The focus was on understanding the distribution and relationships of features within the Iris dataset.
* **Techniques:** Used Scatter plots for feature relationships and Histograms for density analysis.
* **Outlier Detection:** Applied Box plots to identify anomalies, specifically in the `sepal_width` feature.

### 2. Regression & Forecasting (Task 02 & 06)
Applied both Linear and Advanced Gradient Boosting models to predict numerical values.
* **Stock Prediction:** Leveraged the `yfinance` API to fetch real-time Tesla (TSLA) data. Achieved a high **98.6% R² Score** using Linear Regression.
* **House Pricing:** Implemented the **XGBoost Regressor**. Applied advanced preprocessing including Feature Scaling and Outlier removal using **Z-Score** to achieve a near-perfect **0.999 R² Score**.

### 3. Classification & Medical Analytics (Task 03)
Developed a diagnostic model to predict the risk of heart disease based on clinical parameters.
* **Evaluation:** Validated the model using **Confusion Matrix** and **ROC-AUC** scores to ensure medical reliability.
* **Insights:** Performed Feature Importance analysis, revealing that `Max_heart_rate` and `sex` are critical predictors for heart conditions.
