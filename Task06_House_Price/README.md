🏠 House Price Prediction using XGBoost RegressorThis project focuses on predicting residential property prices by analyzing various structural and locational features—such as square footage, room counts, and neighborhood metrics—using advanced gradient boosting techniques.📋 Project OverviewPredicting real estate value is a fundamental yet complex challenge in data science. In this project, I analyzed a comprehensive housing dataset to identify the primary drivers of property value and developed a high-performance regression model to provide accurate price estimations.Key Objectives:Data Preprocessing: Standardizing numerical features like SquareFeet and LotSize for model stability.Feature Engineering: Implementing a custom Location_Score using target encoding on Zip Codes to capture regional value trends.Outlier Management: Utilizing Z-Score analysis to identify and handle statistical anomalies.Advanced Modeling: Leveraging the XGBoost (Extreme Gradient Boosting) algorithm for its superior predictive power and speed.Rigorous Evaluation: Measuring model reliability using MAE, MSE, RMSE, and $R^2$ Score.🛠️ Tech Stack & LibrariesLanguage: Python 🐍Core Libraries: * Pandas & NumPy: For robust data manipulation and mathematical operations.XGBoost: For implementing the gradient boosting framework.Scikit-Learn: For data scaling, dataset splitting, and performance evaluation.Matplotlib: For generating insightful visualizations of price trends.📊 Methodology1. Preprocessing & ScalingFeature Standardization: Applied StandardScaler to SquareFeet and LotSize, ensuring these large-scale variables do not bias the model.Neighborhood Encoding: Since Zip Codes are categorical, I calculated a Location_Score based on the mean price per area. This allows the model to treat location as a continuous, meaningful variable.Data Integrity: Conducted a Z-Score check (threshold > 3) to ensure the training data was free from extreme noise and errors.2. Model ArchitectureI configured the XGBRegressor with the following optimized hyperparameters:n_estimators: 100learning_rate: 0.1max_depth: 53. Performance ResultsThe model demonstrated exceptional precision on the testing data:Mean Absolute Error (MAE): ~3,552.60Root Mean Squared Error (RMSE): ~4,917.11$R^2$ Score: 0.9996 (Indicating a near-perfect correlation between actual and predicted values)📈 Visualizing AccuracyThe Actual vs. Predicted scatter plot shows a very tight alignment along the 45-degree regression line. This confirms that the model has successfully captured the complex relationships between property features and market prices.
