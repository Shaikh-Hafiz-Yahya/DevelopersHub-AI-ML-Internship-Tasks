🏠 House Price Prediction using XGBoost RegressorThis project focuses on predicting residential property prices by leveraging advanced regression techniques and analyzing key features such as square footage, room counts, and location-based metrics.📋 Project OverviewPredicting real estate prices is a fundamental yet complex challenge in machine learning. In this project, we analyzed a comprehensive housing dataset to identify primary price drivers and developed a high-performance regression model to estimate property values with precision.Key Objectives:Feature Preprocessing: Implementing numerical scaling for variables like SquareFeet and LotSize.Feature Engineering: Developing a Location_Score through target encoding of Zip Codes to capture neighborhood value.Outlier Detection: Utilizing Z-Score analysis to identify and manage data anomalies.Model Training: Leveraging the XGBoost (Extreme Gradient Boosting) algorithm for state-of-the-art prediction accuracy.Performance Evaluation: Quantifying error margins using MAE, MSE, and RMSE metrics.🛠️ Tech Stack & LibrariesLanguage: Python 🐍Core Libraries: * Pandas & NumPy: For robust data manipulation and numerical operations.XGBoost: For implementing high-efficiency gradient boosting regression.Scikit-Learn: For data preprocessing, train-test splitting, and evaluation.Matplotlib: For visualizing the correlation between actual and predicted prices.📊 Methodology1. Data Preprocessing & ScalingStandardization: Features like SquareFeet and LotSize were normalized using StandardScaler to ensure all inputs contribute equally to the model's weight updates.Location Engineering: Since Zip Codes are high-cardinality categorical data, we engineered a Location_Score based on mean pricing per area to improve the model's spatial awareness.Outlier Management: We applied a Z-Score filter (threshold > 3) to ensure the model remains robust against extreme data points.2. Model ConfigurationThe XGBRegressor was configured with the following hyperparameters to balance bias and variance:n_estimators: 100learning_rate: 0.1max_depth: 53. Performance MetricsThe model demonstrated exceptional predictive power on the test set:MAE (Mean Absolute Error): ~3,552.60RMSE (Root Mean Squared Error): ~4,917.11$R^2$ Score: 0.9996 (Indicating a near-perfect fit to the data).📈 Results VisualizationThe Actual vs. Predicted scatter plot displays a tight alignment along the 45-degree identity line. This confirms that the XGBoost model successfully captured the non-linear patterns and interactions within the housing dataset.
