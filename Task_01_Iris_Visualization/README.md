# Task 1: Exploring and Visualizing the Iris Dataset

### 🎯 Objective
The primary objective of this task was to perform Exploratory Data Analysis (EDA) on the Iris dataset. This involved loading the data, conducting statistical summaries, and visualizing distributions to understand feature relationships and data patterns.

### 📊 Dataset Details
- **Name**: Iris Dataset
- **Format**: CSV / Seaborn Built-in
- **Features**: Sepal Length, Sepal Width, Petal Length, Petal Width.
- **Target Variable**: Species (Setosa, Versicolor, Virginica).

### 🛠️ Key Steps Performed
1. **Data Inspection**: Used `pandas` (`.info()`, `.describe()`, and `.head()`) to understand the dataset's shape, data types, and summary statistics.
2. **Exploratory Data Analysis (EDA)**:
   - **Scatter Plots**: Implemented using `Seaborn` to visualize the relationship between features.
   - **Histograms**: Created to analyze the value distribution of individual features.
   - **Box Plots**: Used to identify potential outliers and understand the spread of data.
3. **Libraries Used**: `Pandas` for data manipulation, `Matplotlib` and `Seaborn` for high-quality data visualization.

### 📈 Key Results and Findings
- **Correlation**: A strong positive correlation was observed between **Petal Length** and **Petal Width**.
- **Class Separation**: Visualizations clearly showed that the **Setosa** species is easily separable from the other two species based on petal dimensions.
- **Data Quality**: The dataset is well-balanced with no significant outliers affecting the overall distribution.

### 💻 Skills Demonstrated
- Data Loading and Inspection (Pandas)
- Descriptive Statistical Analysis
- Data Visualization (Seaborn & Matplotlib)
